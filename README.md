# Proxmox + LXC + Immich - A selfhosted personal media storage
## Overview
The project is an attempt to solve the challenges that my personal device provided in the form or quickly running out of unexpandable internal storage. Google Photos/iCloud services provide easy to use facility and security up until free data quote is reached (about 5 GB - 15 GB), which is never enough. It also means, these tech companies have my personal data with all details which even I dont know. 
This project lets me explore the world of self managed storage solutions in the form of **Proxmox** and using **Linux containers** to manage the open source application - **Immich**.
